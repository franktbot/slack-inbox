# FrankBot Slack Inbox Aggregator (PRJ-000005)

Read-only Slack middleware + OpenClaw skill for inbox-style scanning of one Slack workspace.

## Permissions
- ✅ Read messages (conversations.list, conversations.history, conversations.replies)
- ❌ Post / react / pin / delete
